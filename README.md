# layout_practice
## 這個是什麼？
因為覺得自己切版太弱所以決定每週練習一次切版，希望藉由每次的練習及記錄讓自己的切版能力變強！
## 目前規劃
- 以 [The Odin Project 中的 HTML / CSS 的作業為目標](https://www.theodinproject.com/courses/html-and-css) 每週練習一次切版，如果時間不允許，可以找其他簡單的範例
- 目前規劃一週一次，不允許的話就兩週一次（要真的不行才可以）
